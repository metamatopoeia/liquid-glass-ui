# @metamatopoeia/liquid-glass-ui

## 1.3.0

### Minor Changes

- [`9277018`](https://github.com/metamatopoeia/liquid-glass-ui/commit/92770182a16f5ca06bbe3f57b0ddedbb1839cfdb) Thanks [@sds-smith](https://github.com/sds-smith)! - introduce SpeedDial component, deprecate AlertDialog component, initiate changelog

  Add Liquid Glass styled SpeedDial and SpeedDialAction components to the navigation UI.

  Features:

  - Support for multiple dial directions (up, down, left, right)
  - Controlled and uncontrolled open state management
  - Liquid glass visual physics engine integration via CSS modules
  - Accessibility support including keyboard navigation and reduced transparency media queries
  - Deprecate AlertDialog component with migration warning to Dialog
